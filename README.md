## Addis Entrance Hub — Mobile App Release

We are excited to introduce the **Addis Entrance Hub Mobile Application**, a fully functional mobile version of the Addis Entrance Hub platform designed to help students prepare efficiently for entrance examinations anytime and anywhere.

This release delivers a smooth, fast, and user-friendly experience, bringing the core features of the website directly to Android devices.

---

##  About the App

**Addis Entrance Hub** is an educational platform that supports students preparing for entrance exams by providing organized learning resources, practice materials, and accessible study tools through a modern mobile interface.

The mobile app is built using **Flutter**, ensuring high performance and a consistent user experience.

---

##  Features

* User-friendly and responsive mobile interface
*  Full website functionality available on mobile
* Secure authentication system
* Study materials and learning resources access
*  Practice and preparation support tools
*  Fast navigation and optimized performance
*  API-based backend integration

---

## Tech Stack

* **Framework:** Flutter (Dart)
* **Backend:** REST API
* **Version Control:** Git & GitHub
* **Platform:** Android

## Installation

1. Download the APK file from the **Assets** section below.
2. Enable *Install from Unknown Sources* on your Android device.
